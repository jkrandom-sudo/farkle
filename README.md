# Farkle / 掷骰子

终端版 Farkle —— 经典推骰子游戏, 你 vs 电脑 AI, 先达到目标分者胜。  
A terminal Farkle (push-your-luck dice game). Race the AI to the target.

## 特性 / Features

- 完整经典 Farkle 计分: 单 1/单 5/三同 (含四同/五同/六同)/顺子/三对/两个三同
- "热骰" 重投全部 6 颗
- 三档 AI 难度: 简单/普通/困难 — 影响落袋阈值与冒险倾向
- 三档目标分: 3000/5000/10000
- 中英双语界面
- 自动选择最高得分组合 (`k`)
- 终端铃声音效 (Farkle 双响, 热骰三响, 胜利三响)
- 持久化设置与 Top 10 排行榜

## 启动 / Run

需要 Python 3.8+, 不依赖任何第三方库:

```sh
python3 game.py
```

## 计分 / Scoring

| 组合 | 分数 |
|------|------|
| 单 1 | 100 |
| 单 5 | 50 |
| 三 1 | 1000 |
| 三 N (N=2..6) | N × 100 |
| 四同 | 2 × 三同 |
| 五同 | 4 × 三同 |
| 六同 | 8 × 三同 |
| 顺子 1-2-3-4-5-6 | 1500 |
| 三对 | 1500 |
| 两个三同 | 2500 |

**注**: 保留的骰子集合必须每颗都计入计分组合 (没有"剩余"散粒)。

## 操作 / Commands

每次投骰后:

| 输入 | 含义 |
|------|------|
| `1 5` 或 `1,5` | 保留这些点数的骰子 |
| `k` | 自动选择最高得分组合 |
| `b` | 落袋, 累计到总分 |
| `r` | 投剩余骰子 (热骰会重投全部 6 颗) |
| `q` | 放弃本局 |

**重要**: 每轮投骰至少要保留一颗计分骰子才能继续投或落袋。  
若投出无任何计分组合 → **Farkle**, 本回合分数清零, 轮到 AI。

## 测试 / Tests

```sh

```

包含 81 个单元测试: 计分引擎 (所有组合), Farkle 判定, 最佳子集搜索, AI 决策, 输入解析, 玩家/AI 回合, 主菜单, 设置/排行榜/i18n/音效。
